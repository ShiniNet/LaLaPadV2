# LaLaPad V2

「キーボードから手を離さない」ための、完全統合型ポインティングキーボード。

LaLaPad V2 は、
独自設計のダブルトラックパッド（IQS9151）を搭載した
無線対応 40% 分割キーボードです。

トラックパッドに触れるだけでカーソル・スクロール・左右中クリック・ジェスチャー操作が完結します。

Double Trackpad Design

左右両側に高機能トラックパッドを搭載。
両側ともフル機能のマウスとして使用可能
左利きユーザーや液タブ併用時のポインティング用途にも最適

もうモジュールを左右で付け替えたりする必要はありません。

Gesture & Smooth Control

マルチフィンガージェスチャー対応
慣性カーソル / 慣性スクロール
ビルド不要の動的調整

用途に合わせたカスタマイズが可能です。
